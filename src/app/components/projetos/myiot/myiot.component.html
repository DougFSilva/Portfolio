MyIot

Plataforma para controle de dispositivos IOT

A plataforma MyIot tem como objetivo gerenciar dispositivos Interner das coisas, com possibilidade de realizar comandos em dispositivos discretos (liga-desliga) para por exemplo ligar uma lâmpada, um ventilador, um motor, etc... Também oferece a possibilidade de comandar dispositivos de saída analógica, para realizar por exemplo, o controle da velocidade de um motor e a intensidade de iluminação de uma lâmpada. Com a plataforma também é possível registrar medições de dispositivos de medição e verificar os registros por meio de gráfico e tabela. Os estados e medições dos dispositivos são verificados em tempo real na tela.

O backend é desenvolvido em java com o framework Spring, utilizando os módulos:

  • Spring Boot
  • Spring Security
  • Spring Web
  • Spring WebSocket

A autenticação é realizada através do método JWT(Json Web Tokens) e o banco de dados utilizado é o MongoDb

O frontend é uma single page application desenvolvida com o FrameworkAngular

A comunicação entre o servidor e os dispositivos são realizados por meio de protocolo Mqtt, funcionando muito bem com oESP32, ESP8266, Arduíno e afins

FUNCIONAMENTO

Primeiramente o usuário faz seu cadastro no sistema e aguarda até que seja aprovado por algum usuário ADMIN. Após aprovado, o novo usuário possuirá um perfil do tipo SILVER_USER, possibilitando-o a cadastrar até 6 dispositivos discretos, 6 dispositivos analógicos e 4 dispositivos de medição. O perfil do usuário pode ser alterado por um usuário ADMIN, tornando-o GOLD_USER ou até mesmo um ADMIN. Usuário com perfil GOLD_USER podem cadastrar 12 dispositivos discretos, 12 dispositivos analógicos e 8 dispositivos de medição. Usuário com perfil ADMIN pode cadastrar a mesma quantidade de dispositivos que o usuário GOLD_USER.

Já aprovado, o usuário faz login no sistema e então cadastra os os dispositivos desejados. Com isso, para cada dispositivo cadastrado é gerado um tópico no broker MQTT, no qual somente o usuário poderá publicar e se inscrever através do username (email do usuário) e password que pode ser acessado pelo usuário na tela "Minha conta"

Para encontrar o tópico gerado no broker MQTT basta acessar o dispositivo desejado, clicar no ícone de menu do mesmo e cliar em copiar tópico MQTT para que o tópico seja copiado para a área de transferência. Então pressionado CTRL + V o tópico será colado onde desejado.

O tópico gerado copiado serve somente para publicar e receber dados do Broker MQTT, para que seja salvo no banco de dados deve-se inserir /persist no final do tópico, por exemplo:

  • Se o tópico copiado foi iot/DiscreteDevice/115s4sa5adrg6sd88, para que os dados sejam salvos o tópico a se publicar deverá ser iot/DiscreteDevice/115s4sa5adrg6sd88/persist
